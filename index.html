Apollo

Your musical muse for a new age

What Does Apollo Do?

Apollo lets you get automated information about your favorite songs, like their tempo, loudness, and Spotify ratings of acousticness, danceability, and more.

An example of the output from Apollo for Gooey by Glass Animals and Outlaws by Alessia Cara

How Does Apollo Work?

Using the Spotify API, Apollo combines data returned from the "Get a Track" and "Get Audio Features for a Track" queries to get as much information as possible about a given song. The data returned from the Spotify API is then graphed and displayed. Apollo needs the Spotify URI or the track ID to run these API queries, which you can retrieve by right clicking a song in Spotify and clicking "Copy Spotify URI".
